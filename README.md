# Subtitle Production Tutorial
**CheeseAni 的字幕制作教程**  

正在持续更新中……


## 关于本教程
### 教程目标
本教程旨在让跟多人能更轻松、更简单地学会字幕制作相关知识  
让更多人能够参与字幕制作

### 学习路径
本教程的学习路径是非线性的  
首先会将每个环节大致介绍一遍  
然后再逐渐深入讲解  
旨在让新手也能快速入门，能够一人走完全部流程，入门后再逐渐加深学习  
本教程不含外语相关知识

### 教程内容
本教程涉及以下内容
* 字幕组工作流程
* Aegisub等相关工具使用
* ASS语法与编辑技巧
* 视音频相关知识
* 流程管理相关技巧
